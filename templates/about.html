About KindEye

KindEye is your 24/7 emotional support platform powered by AI technology. We aim to cure loneliness and provide empathetic conversations anytime, anywhere.

Purpose

Our mission is to provide a healing space where technology meets empathy. KindEye is designed to offer real-time emotional support through AI-driven conversations that understand you deeply.

Technologies Used

Technology Description Logo
Flask (Backend) Flask is a lightweight web framework used for the backend of KindEye. It handles requests, processes data, and interacts with the SQLite database to store user information and conversation histories. It also integrates seamlessly with Ollama for AI processing.
Ollama (AI Processing) Ollama is the AI engine behind KindEye. It processes and understands user queries, providing intelligent, empathetic responses. Ollama uses natural language processing (NLP) to make conversations feel more human-like and emotionally supportive.
SQLite (Database) SQLite is a lightweight relational database used by KindEye to store and retrieve data. It stores user conversations, settings, and preferences, providing persistent storage to personalize interactions.
Bootstrap (Frontend Design) Bootstrap is a front-end framework that allows KindEye's interface to be responsive and visually appealing across all devices. It provides components like grids, buttons, and forms, enabling quick and efficient design.
Python Python is the main programming language used in KindEye for both backend logic and integration with the AI engine (Ollama). It simplifies tasks and offers excellent libraries for data processing, web development, and AI integration.
CSS (Designing) CSS is used to style KindEye's interface, ensuring it is visually attractive and consistent. It controls the layout, colors, fonts, and overall design of the web pages.
HTML (Structure) HTML is the backbone of KindEye's web interface. It defines the structure of the web pages, organizing content such as text, images, buttons, and other elements.
JavaScript (Speech Synthesizer) JavaScript is used to create dynamic and interactive elements on the frontend, including the integration of a speech synthesizer that allows AI-driven voice responses for a more engaging and immersive user experience.
